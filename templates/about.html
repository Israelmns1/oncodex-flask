About OncoDex

OncoDex is a web app where users can upload, preview, and analyse datasets stored in CSV files, providing simple summary statistics and visualisations for quick data exploration.

Features

  • Upload multiple CSV files containing experimental or clinical data
  • Preview datasets in a clean, tabular format
  • View key summary statistics to understand data distribution
  • Visualise numeric columns with bar plots showing means and standard deviations

How to Use

  1. Navigate to the Upload page to add your CSV files
  2. Head to the Analyse page to select a file and explore its contents
  3. Review summary statistics and the mean Â± standard deviation plots for numeric data

OncoDex is built with Flask, Pandas, and Matplotlib.
